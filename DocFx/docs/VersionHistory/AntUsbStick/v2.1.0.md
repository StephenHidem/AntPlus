# Version 2.1.0
## Release Date: June 12, 2024
- OpenedFrameType and OpenedPortType have been removed in this release.
- The GetSerialString and GetProductDescription methods in AntUsbRadio have been marked as obsolete.
- Overloads in the interfaces have been removed in favor of optional arguments with default values.
The GetDeviceCapabilities method may require rework in implementations.
