      Version 3.0.0.0

      Release 3.0.0.0 was released on December 29, 2023.

      Changes in This Release

      There are breaking changes in the AntRadioInterface and AntPlus class libraries. You will need to refactor some of your code that has dependencies on these libraries.

      AntRadioInterface 2.0.0

      AntRadio - some interfaces have been changed to Task return types. Others likely to follow in the future.

      AntPlus 3.0.0

      1. CommonSpeedCadence - bug fix handling default data page 0.
      2. Treadmill - added None to CapabilityFlags enum.
      3. AntDevice - fixed log warning bug in SendExtAcknowledgedMessage method.
      4. AntDeviceCollection - ctor has been modified.
      5. BicyclePower/Parameters - all methods are now async calls.
      6. Tracker - the asset is removed from the Assets collection if the Asset.AssetStatus.RemoveAsset flag is set.

      AntUsbStick 2.0.0

      AntRadio - implementation uses the async task pattern defined in the AntRadioInterface.

      AntMulticastServer 2.0.0

      Aligned the AntMulticastServer example more closely to the DI pattern.

      WpfUsbStickApp 2.0.0

      Implementation modified based on the AntRadioInterface.

      NEW! .NET MAUI example with gRPC

      Implements a .NET MAUI client application utilizing gRPC for remote procedure calls. Details are located here.
