# Version 2.0.0
## Release Date: July 7, 2023
Breaking Changes:
- Renamed BicyclePower PowerOnlySensor to PowerSensor.
- AssetTracker class renamed to Tracker.
- AssetTracker Asset latitude and longitude are now reported in decimal degrees.
- FitnessEquipment class renamed to Equipment.
- Namespaces modified - the following classes are rooted in DeviceProfiles:	Geocache, HeartRate, MuscleOxygen,
StrideBasedSpeedAndDistance, UnknownDevice.
- All device profiles require an ILogger in their ctor. The AntDeviceCollection requires an ILoggerFactory which
is used to generate ILogger's for the device profiles.
