      WpfUsbStickApp

      The WpfUsbStickApp example demonstrates connecting to a Garmin/Dynastream ANT USB stick, enumerating ANT devices, and interacting with an ANT device. The ANT USB stick is initialized for receive scan mode. The ANT+ class library AntDeviceCollection serves the role of enumerating ANT devices and interacting with them.

        Note

      This project uses the AntUsbSick project. Therefore, the platform target is set to x86. This is due to the native DLLs that are a part of this project.

      Notes About the AntDeviceCollection

      The AntDeviceCollection is the focal point of this example. It's an observable collection that the application subscribes to via XAML binding. An ANT device can be selected and the device can be interacted with.
